# 11.2 人工智能技术概述

## 1. 本节核心思想（100字以内）

人工智能旨在让机器模拟人类智能，通过机器学习、深度学习等技术实现感知、理解、推理和决策。重点掌握人工智能的发展历程、弱AI与强AI的区别、六大关键技术，以及机器学习的分类和典型应用。

---

## 2. 考试高频考点（尽量精简）

### （1）弱人工智能 vs 强人工智能 ★★★★★
- 弱AI：专用智能，无自主意识（当前所有AI）
- 强AI：具备自主意识和通用智能（尚未实现）

### （2）人工智能发展历程 ★★★★☆
- 1950：图灵测试
- 1956：达特茅斯会议（AI诞生）
- 1968：专家系统 DENDRAL
- 20世纪70年代：AI寒冬
- 1997：Deep Blue 战胜国际象棋冠军
- 2006：深度学习兴起
- 2016：AlphaGo 战胜李世石

### （3）人工智能六大关键技术 ★★★★★
- NLP（自然语言处理）
- 计算机视觉（CV）
- 知识图谱（KG）
- 人机交互（HCI）
- VR/AR
- 机器学习（ML）

### （4）机器学习四大分类 ★★★★★
- 监督学习：有标签
- 无监督学习：无标签
- 半监督学习：少量标签
- 强化学习：奖励机制

### （5）传统机器学习 vs 深度学习 ★★★★☆
| 传统机器学习 | 深度学习 |
|--------------|----------|
| 人工提取特征 | 自动学习特征 |
| 数据需求少 | 数据需求大 |
| 算力要求低 | GPU算力高 |
| 可解释性较强 | 可解释性较弱 |

### （6）常见算法（了解）
- 传统：逻辑回归、决策树、SVM、KNN、贝叶斯
- 深度学习：CNN、RNN

### （7）机器学习应用（了解）
- 广告推荐
- 图像识别
- 语音识别
- 自动驾驶
- 金融风控
- 医疗诊断
