# 11.1 信息物理系统（CPS）技术概述

## 1. 本节核心思想（100字以内）

信息物理系统（CPS）通过融合感知、计算、通信和控制技术，实现物理空间与信息空间的深度协同，形成“感知→分析→决策→执行”的闭环优化体系。其建设遵循单元级→系统级→SoS级逐步演进路径，以实现资源优化配置和智能化运行。

---

## 2. 考试高频考点（尽量精简）

### CPS定义与本质
- CPS = Cyber-Physical Systems（信息物理系统）
- 本质：数据驱动的闭环赋能体系
- 核心闭环：
  - 状态感知
  - 实时分析
  - 科学决策
  - 精准执行

### CPS三层架构
#### 单元级CPS
- 最小单元
- 功能：感知、计算、控制、通信

#### 系统级CPS
- 多个单元级组成
- 功能：互联互通、协同控制、监视诊断

#### SoS级CPS
- 多个系统级组成
- 功能：数据汇聚、大数据分析、智能服务

### CPS四大核心技术要素
- 一硬：感知与自动控制
- 一软：工业软件
- 一网：工业网络
- 一平台：工业云与智能服务平台

### CPS典型应用场景
- 智能设计
- 智能生产
- 智能服务
- 智能应用

### CPS建设路径
1. CPS体系设计
2. 单元级CPS建设
3. 系统级CPS建设
4. SoS级CPS建设
