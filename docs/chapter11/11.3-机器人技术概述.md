# 11.3 机器人技术概述

## 1. 本节核心思想（100字以内）

机器人技术介绍了机器人的概念、三代发展历程、机器人4.0核心技术及分类方式。机器人正从"示教执行"向"智能感知、自主决策、云边端协同"演进，广泛应用于工业、服务和特殊领域。

---

## 2. 考试高频考点（尽量精简）

- Robot原意：苦力、奴隶
- 三代机器人：
  - 第一代：示教再现型
  - 第二代：感觉型
  - 第三代：智能型
- 机器人4.0五大核心技术：
  - 云-边-端协同
  - 持续学习与协同学习
  - 知识图谱
  - 场景自适应
  - 数据安全
- 两种分类方式：
  - 控制方式：操作、程序、示教再现、智能、综合
  - 应用领域：工业、服务、特殊领域

---

## 3. 记忆口诀（尽量精简）

**一教二感三智能；云学知场安；操程教智综；工服特。**

- 一教二感三智能：机器人三代发展
- 云学知场安：机器人4.0五大核心技术（云边端、学习、知识图谱、场景自适应、安全）
- 操程教智综：按控制方式分类
- 工服特：按应用领域分类
