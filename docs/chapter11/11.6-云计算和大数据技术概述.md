# 11.6 云计算和大数据技术概述

## 1. 本节核心思想（100字以内）

云计算通过按需提供计算、存储和平台等资源，实现资源共享与弹性扩展；大数据技术围绕海量、多样、高速数据进行采集、存储、处理、分析和应用，挖掘数据价值，两者共同支撑现代信息系统和智能应用的发展。

---

## 2. 考试高频考点（尽量精简）

### 云计算
- **三种服务模式（★★★★★）**
  - SaaS：软件即服务
  - PaaS：平台即服务
  - IaaS：基础设施即服务
- **特点**
  - 灵活性：**IaaS > PaaS > SaaS**
  - 方便性：**SaaS > PaaS > IaaS**
- **四种部署模式**
  - 公有云
  - 私有云
  - 社区云
  - 混合云
- **发展对应关系**
  - 虚拟化 → IaaS
  - 分布式计算 → PaaS
  - 软件服务 → SaaS

### 大数据
- **4V特征（★★★★★）**
  - Volume（海量）
  - Velocity（高速）
  - Variety（多样）
  - Value（价值）
- **处理流程**
  - 获取 → 清洗 → 集成 → 分析 → 展示（解释）
- **典型应用**
  - 制造、服务、交通、医疗

## 3. 记忆口诀（尽量精简）

### 云计算
- **三层服务：** 基软平（**基**础、**平**台、**软**件）
- **灵活性：** **I>P>S（越底层越灵活）**
- **方便性：** **S>P>I（越上层越省事）**
- **四种部署：** **公社私混**
- **发展历程：** **虚→基，分→平，软→软**

### 大数据
- **4V：** **大量、多样、高速、价值**
- **处理流程：** **采→洗→集→析→展**
- **应用领域：** **制服交医（制造、服务、交通、医疗）**
