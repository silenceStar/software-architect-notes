# 11.5 数字孪生体技术概述

## 1. 本节核心思想（100字以内）

数字孪生体是在数字世界中构建现实对象的实时映射，通过建模、仿真、数字线程等技术，结合物联网、大数据、人工智能等，实现物理实体的监测、预测、优化和控制，广泛应用于智能制造、产业、智慧城市和军事等领域。

---

## 2. 考试高频考点（尽量精简）

- 数字孪生体：现实对象的数字化实时映射
- 发展四阶段：准备→概念→应用→推广
- 三大核心技术：建模、仿真、数字线程
- 顶层框架：MBSE（基于模型的系统工程）
- 外围技术：物联网、云计算、大数据、AI、区块链、边缘计算
- 四大应用：制造、产业、城市、战场
