# 2.7 多媒体

## 1. 本节核心思想（100字以内）

多媒体是利用计算机将文本、图形、图像、音频、动画和视频等多种媒体进行集成、处理、存储和传输，并形成具有交互性的系统。其核心特征是多维化、集成性、交互性和实时性，关键技术包括视音频处理、通信、数据压缩以及VR/AR技术。

## 2. 考试高频考点

* **考点1：媒体的五种分类**
  媒体可分为感觉媒体、表示媒体、表现媒体、存储媒体和传输媒体，区分时要抓住“人的感觉、信息形式、物理设备、存储介质和传输介质”。

* **考点2：多媒体的四大特征**
  多媒体具有多维化、集成性、交互性和实时性，其中交互性表示用户由被动接收信息变为主动控制信息。

* **考点3：多媒体系统的组成**
  多媒体系统由硬件和软件组成；硬件包括多媒体计算机、板卡和外部设备，软件包括驱动程序、操作系统、素材制作工具、著作工具和应用软件等。

* **考点4：视音频技术**
  视频技术包括视频数字化和视频编码，音频技术包括音频数字化、语音处理、语音合成和语音识别。

* **考点5：无损压缩与有损压缩**
  无损压缩前后的数据完全一致，利用数据统计冗余，压缩比较低；有损压缩会丢失人眼或人耳不敏感的信息，压缩比较高且信息不可恢复。

* **考点6：常见多媒体编码标准**
  JPEG主要用于静态图像压缩，MPEG主要用于动态图像压缩，H.26L主要用于可视电话、视频会议等实时视频通信。

* **考点7：VR与AR的区别**
  VR利用计算机创建完整的虚拟环境，使用户沉浸其中；AR是在现实世界中叠加虚拟信息，用户仍能感知真实环境。

* **考点8：VR/AR的四种类型**
  VR/AR可分为桌面式、分布式、沉浸式和增强式，其中分布式强调网络共享，沉浸式强调完全沉浸，增强式强调虚实叠加。

* **易混点：表示媒体与表现媒体**
  表示媒体是图像、声音、视频等信息的表示形式；表现媒体是键盘、摄像机、显示器和音箱等输入输出物理设备。

* **易混点：感觉媒体与表示媒体**
  感觉媒体强调人通过视觉、听觉、触觉等方式感知信息；表示媒体强调信息在计算机中的具体表现形式。

* **易混点：数据压缩与文件压缩**
  数据压缩多用于具有时间性的音频、视频等实时数据；文件压缩主要针对保存在磁盘等介质上的文件。

* **易混点：即时压缩与非即时压缩**
  即时压缩是在数据传输过程中同步压缩，常用于实时音视频；非即时压缩是先完成压缩再进行传输。

* **易混点：VR与AR**
  VR用虚拟环境替代现实环境，强调“沉浸”；AR在现实环境中叠加虚拟内容，强调“增强现实”。

* **易考选择题：键盘和鼠标属于哪类媒体？**
  键盘和鼠标是用于获取信息的物理设备，属于表现媒体，也称显示媒体中的输入媒体。

* **易考选择题：硬盘、光盘和RAM属于哪类媒体？**
  它们用于存储信息的表示形式，属于存储媒体。

* **易考选择题：电缆、光缆和电磁波属于哪类媒体？**
  它们用于传递信息，属于传输媒体。

* **易考选择题：JPEG、MPEG和H.26L分别用于什么？**
  JPEG用于静态图像，MPEG用于动态图像，H.26L侧重实时视频通信。

* **易考选择题：哪种压缩方式的数据不可完全恢复？**
  有损压缩会丢失部分不敏感信息，解压后的数据不能完全恢复为原始数据。

## 3. 记忆口诀

### 口诀

**感表现，存传五；多集交实四特征；视音通压虚增强；静J动M实时H；虚拟沉浸，增强叠真。**

### 解释

* **感表现，存传五**：媒体分为感觉媒体、表示媒体、表现媒体、存储媒体和传输媒体。
* **多集交实四特征**：多媒体四大特征是多维化、集成性、交互性和实时性。
* **视音通压虚增强**：多媒体关键技术包括视音频、通信、数据压缩、虚拟现实和增强现实。
* **静J动M实时H**：JPEG用于静态图像，MPEG用于动态图像，H.26L用于实时视频通信。
* **虚拟沉浸，增强叠真**：VR强调进入虚拟世界，AR强调在真实世界中叠加虚拟信息。

本节最可能考媒体五种分类、多媒体四大特征、无损与有损压缩、JPEG/MPEG/H.26L标准，以及VR与AR的区别。
